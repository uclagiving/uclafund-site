This plugin removes the "Protected" or "Private" prefix that wordpress adds to Password protected / Private pages.

== Description ==

Remove Protected In Title is a plugin that removes the prefix that wordpress adds on private and password protected pages.
Usage: Install and activate the plugin. Check one of your password protected or private pages to make sure everything works ok.

== Installation ==

Through wordpress admin:

1. Go to Plugins -> Add New
1. Search for "Remove Protected Title"
1. Install the plugin
1. Activate it

Through FTP:

1. Download the plugin
1. Unzip remove-protected-title.zip and upload the folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= How to use? =

After activation the plugin is loaded automatically. It will remove "Protected" and "Private" from all page titles on your site
